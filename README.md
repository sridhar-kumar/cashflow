# 💰 CashFlow Pro — Expense Tracker Dashboard

## 📌 Overview

CashFlow Pro is a responsive expense tracking dashboard built using HTML, CSS, and JavaScript.
It allows users to manage their salary, track expenses, visualize data, and export reports.

---

## 🚀 Features

### 🔹 Core Features

* Add and delete expenses
* Real-time balance calculation
* Input validation for secure data entry

### 🔹 Data Management

* Local Storage support (data persists after refresh)

### 🔹 Visualization

* Interactive doughnut chart using Chart.js
* Displays expense vs balance distribution

### 🔹 Advanced Features (Level 3)

* Currency conversion (INR ↔ USD) using API
* PDF report generation using jsPDF
* Low balance alert system

---

## 🖥️ Tech Stack

* HTML5
* CSS3 (Modern UI with glassmorphism)
* JavaScript (Vanilla JS)
* Chart.js
* jsPDF
* Fetch API
